# Competitor Teardown: Online Rx Eyewear Ordering

Scope: how competitors take a shopper from "I like this frame" to checkout, with focus on configurator flow, Rx intake, pricing, and positioning. Purpose: inform GlassyVision product configurator + Rx intake design.

---

## 1. Zenni Optical

**Configurator flow (7–8 steps, frame-first):**
1. PDP: pick frame, color, size.
2. Click "Buy frame now" → "Select Usage" modal (Single vision distance / reading / progressive / non-Rx / Rx sunglasses).
3. Enter prescription (manual SPH/CYL/AXIS, optional ADD, prism, PD).
4. Choose lens type (Standard 1.50 / mid 1.57 / high-index 1.61 / 1.67 / 1.74).
5. Choose lens package: Basic / Blokz blue-light / Transitions / Sunglasses tint / Polarized.
6. Add-ons: AR coating (often included at higher packages), anti-fog, oleophobic.
7. Review → Cart → Checkout.

**Rx intake:** Manual entry is the default path; image upload and "email my doctor" are secondary. Collected **before payment**. Fields: SPH, CYL, AXIS, ADD, prism (optional), PD (required, with a self-measurement tool — ruler + webcam flow). Client-side validates ranges (e.g., AXIS 1–180). Has "Save Rx to account" and guest save via email link.

**Pricing:** Frames from $6.95. Typical all-in Rx sunglass: $30–$60. Polarized ~$29 upcharge, high-index 1.67 ~$35, Transitions ~$69. Upsells shown as a vertical list with checkbox selection — minimal pre-checking.

**Positioning:** Value leader. Cluttered catalog, low trust signals, but ruthless price transparency. Target: budget-conscious, Rx-literate shoppers.

## 2. Warby Parker

**Configurator flow (frame-first, but funneled through Home Try-On first):**
1. Browse → "Try 5 at home free" CTA or direct PDP.
2. PDP → "Select lenses" → Usage (single vision / progressive / readers / non-Rx).
3. Lens package: Standard Polycarbonate (included) / Light-Responsive (Transitions) / Blue-Light / Sun / Premium Progressives.
4. "Do you have your prescription?" → upload / enter / we'll call your doctor / "Add Rx later."
5. PD entry with explainer.
6. Review → Checkout.

**Rx intake:** All four options offered: upload image, manual entry, doctor contact, "add later" (very important — doesn't block purchase). Rx can be added **after payment** via account link. PD collected with a printable ruler PDF. Clean field labels, SPH/CYL/AXIS/ADD/PD/prism.

**Pricing:** Flat $95 for single-vision including AR + scratch coat + polycarbonate — famously bundled. Progressives +$200, high-index +$30, light-responsive +$100, polarized sun +$50. Predictable, honest-feeling.

**Positioning:** Premium accessible. Editorial photography, serif type, warm storytelling ("buy a pair, give a pair"). The emotional benchmark; not the price benchmark.

## 3. Pair Eyewear

**Configurator flow (base + topper, unique mental model):**
1. PDP: pick "Base Frame" (the Rx carrier).
2. Pick Top Frames (magnetic decorative overlays) — this is the headline interaction, a grid with live preview swapping.
3. Select lens type (single-vision / progressive / readers / non-Rx / sun).
4. Rx entry or upload.
5. Lens upgrades (light-responsive, blue-light, etc.).
6. Cart.

**Rx intake:** Upload, manual entry, "send later" all supported. Collected before payment but skippable. Clean stepper with progress dots. PD optional with default fallback.

**Pricing:** Base frame $60–$95, Top Frames $25 each (often sold in bundles of 2–4). Progressives +$175. Deliberately sticky pricing — upsell is *more toppers*, not more coatings.

**Positioning:** Playful, youth-forward, IP-heavy (Marvel, Sanrio, Disney, NBA collabs). Bright color, rounded type, TikTok-native. **Closest reference to GlassyVision's youth/techie angle** — but more kids/Gen-Z-feminine than techie.

## 4. EyeBuyDirect

**Configurator flow (frame-first, 6 steps):**
1. PDP → "Select prescription."
2. Usage type.
3. Rx entry (manual / upload / saved).
4. Lens index.
5. Lens treatment/package (KODAK lens branding featured heavily).
6. Add-ons (AR, blue-light, anti-fog).
7. Cart.

**Rx intake:** Manual, upload, saved-to-account, "enter later" (skip). Collected before payment. PD with self-measure tool and explainer video. Fields standard.

**Pricing:** Frames $6–$50. Aggressive BOGO promos always on. All-in Rx sunglass ~$45–$80. Upsell wall is long; several coatings pre-recommended but not pre-selected.

**Positioning:** Mid-market, promo-driven, SEO beast. Generic design language — no strong brand personality. Heavy reliance on discount urgency ("Ends in 2h 14m").

## 5. Felix Gray

**Configurator flow (minimal, 4–5 steps):**
1. PDP → "Shop this frame" → Lens selection: Non-Rx Blue-Light / Reading / Single Vision / Progressive / Sun.
2. If Rx: strength selection (readers) or Rx entry.
3. Lens material (most bundled).
4. Cart.

**Rx intake:** Manual or upload. Collected before payment. Rx is almost an afterthought — most of their traffic buys non-Rx blue-light computer glasses. No PD required for plano blue-light.

**Pricing:** Flat $95 computer glasses, $145–$195 Rx. Very Warby-like flat pricing. Blue-light coating (their signature) is always included, not an upsell.

**Positioning:** Tech wellness. Muted palette, editorial photography of tired developers, wellness copy ("screen fatigue, solved"). Clean minimalist UI. **Second-closest reference for techie angle** — their tone is the adult version of what we want.

## 6. GlassesUSA

**Configurator flow (frame-first, long — 8+ steps):**
1. PDP → "Select lenses."
2. Usage type.
3. Lens type (single/bifocal/progressive).
4. Material/index.
5. Coating bundle (Basic / Premium / Elite — **pre-selected to Premium**).
6. Additional add-ons.
7. Rx entry (manual / upload / call doctor / send later).
8. Cart, with sticky upsell modal.

**Rx intake:** All four options. Collected before payment. Fields standard. Validation minimal.

**Pricing:** Frames $39–$200+. Branded designer options (Ray-Ban, Oakley) mixed with generic. Heavy BOGO and first-order discounts. Final price often 2–3x frame list due to coating bundles.

**Positioning:** Department-store-online. No strong voice. Uses urgency countdowns, exit-intent modals, and a pre-selected mid-tier coating bundle that nudges ~$80 onto every order. **Worst offender on dark patterns** in this list.

## 7. Lensabl

**Configurator flow (lens replacement — inverse model):**
1. "Send us your frame" flow: pick lens type → Rx entry → ship frame in, lenses mailed back.
2. Also sells new frames with Rx, but lens-replacement is the hook.
3. Steps: Lens type → Material → Coatings → Rx → Checkout.

**Rx intake:** Manual, upload, doctor contact. Before payment. Standard fields.

**Pricing:** Lens replacement from $77 single-vision, $297 progressive. Expensive vs. new-frame competitors. Justified as "keep the frame you love."

**Positioning:** Service-first, quiet brand. The existing GlassyVision scaffold copied their step structure — worth revisiting. Lensabl's flow is good for lens-only but over-complex for new frames.

## Cross-Cutting Themes

**Configurator pattern:** 6 of 7 competitors use **frame-first**. Only Lensabl inverts it (because the frame is a given). Shopper mental model: "I fell in love with a frame, now make it work for my eyes." Frame-first wins.

**Rx-before-or-after-payment:** 5 of 7 collect Rx **before payment** by default. Warby and Pair are the progressives — they allow skip-and-add-later. This is the single biggest conversion lever: forcing Rx entry mid-funnel loses cart abandoners who don't have their Rx handy. Best practice: collect *optionally* before payment, but **never block checkout** on Rx — send a reminder link post-purchase.

**PD (pupillary distance):** The universal friction point. Zenni, Warby, EBD all have self-measure tools (webcam + card or printable ruler). Felix Gray skips PD for blue-light-only. Default-friendly fallback (e.g., 63mm men / 61mm women) with a "we'll double-check before manufacturing" safety net is the pragmatic play.

**Upsell UX:** Warby and Felix Gray bundle aggressively into a flat price — feels honest. Zenni itemizes — feels transparent. GlassesUSA pre-selects Premium — feels like a trap. EyeBuyDirect does promo-stacking — feels cheap but effective.

**Progress visibility:** Pair and Warby have clear stepper bars. Zenni and GlassesUSA do not — users get lost. Steppers measurably reduce abandonment.

## Steal List (concrete UX patterns to copy)

1. **Pair's live Top Frame swap preview** — grid of variants with instant hero image update. Apply this to frame colors and lens tints.
2. **Warby's "Add Rx later" option** — never block checkout on Rx; send a secure link post-purchase. Biggest cart-recovery lever in category.
3. **Warby's flat-bundle pricing** — one headline number includes AR + scratch + polycarbonate. Reduces decision fatigue; feels premium.
4. **Zenni's PD self-measure webcam tool** — gives budget-tier users agency. Builds trust with Rx-literate shoppers.
5. **Felix Gray's minimal 4-step configurator** — our default for techie audience. Hide advanced coatings behind "Customize" disclosure.
6. **Warby's Home Try-On (5 frames free)** — signature trust builder. Expensive operationally but unmatched as a moat. For GlassyVision, consider a limited "try 3" program or an AR try-on alternative.
7. **Pair's progress stepper with dots** — clean, mobile-first, makes the flow feel short even when it isn't.
8. **EyeBuyDirect's "saved prescriptions" per account** — reduces friction on re-order, critical for LTV.
9. **Warby's printable PD ruler PDF** — low-tech fallback when webcam tool fails. Zero-effort to build, high-trust.
10. **Zenni's client-side Rx range validation** — catches typos (AXIS 900 instead of 90) before submission. Saves support tickets.

## Gap List (what no one does well — opportunities for GlassyVision)

1. **Drop culture for eyewear.** No one runs scheduled limited drops ("Friday 5pm: 200 pairs"). Pair does collabs but not drops. Massive untapped lever for trendy/techie youth — Supreme/Telfar mechanics applied to frames.
2. **Honest Rx pricing without upsell trap patterns.** Warby is honest but expensive; Zenni is cheap but cluttered. Gap: techie-clean UI at Zenni-adjacent prices.
3. **Rx-as-a-saved-profile across devices.** No competitor lets you scan your Rx with your phone camera (OCR) and store it as a reusable profile across future orders and frames. Huge quality-of-life win.
4. **Developer/creator-native features.** Dark mode UI, glare-test demos, a "which frame fits my monitor setup" quiz, font-rendering-quality copy for blue-light frames. No one speaks to the techie audience in their language.
5. **Transparent India-fulfillment story.** Zenni is Chinese-made but hides it; we should own India fulfillment as a narrative (craft, hand-finishing, lower carbon than China-to-US) — this is a positioning play nobody else uses.
6. **Real subscription / refresh program.** Pair sells toppers but no one does a "new frame every 6 months, keep lenses" program. Could be a meaningful LTV model.
7. **Open API / Shopify-native Rx object.** No competitor treats Rx as a first-class portable data object. Developer angle: export Rx as JSON/PDF for doctors.
8. **Sunglass-first Rx flow.** Most flows funnel you through clear-lens assumptions. For GlassyVision's sunglasses emphasis, make tint/polarization/mirror the first lens question.

## Avoid List (dark patterns and bad UX not to copy)

1. **Pre-selected mid/premium coating bundles** (GlassesUSA, EyeBuyDirect). Always default to the cheapest viable option; make upgrades opt-in.
2. **Fake urgency countdowns** ("Offer ends in 2h 14m") — EyeBuyDirect / GlassesUSA. Erodes trust the moment a user notices they reset.
3. **Hidden "required" fields that appear after clicking Next** (Zenni). Disclose all required fields up front.
4. **Forcing Rx entry before showing final price** (GlassesUSA). Show the price before the Rx wall.
5. **Stock photo reviews / unverified star counts** (GlassesUSA). Only show reviews tied to verified orders.
6. **Modal-bombing on checkout** (exit-intent discount popups) — EyeBuyDirect is the worst offender. Creates a Pavlovian "always wait for a popup" behavior that trains users to not pay list price.
7. **Bundling AR coating as a separate line item** when it's industry-standard (Zenni). Users feel nickel-and-dimed.
8. **Over-long configurators** (8+ steps, GlassesUSA). Each extra step costs ~10% of cart completion.
9. **"Email your doctor" as an only Rx option** — creates delay, kills conversion. Must always be one option among many.
10. **Asking for prism/ADD/pantoscopic tilt on every order** when only ~5% of shoppers need them. Hide behind an "Advanced" disclosure.

## Recommendation Summary

**Closest reference:** Pair Eyewear (for mental model + youth energy) blended with Felix Gray (for techie minimalist tone).

**Default configurator shape for GlassyVision:**
Frame → Lens usage → (optional Rx, non-blocking) → Lens tint/coatings bundled → Checkout → Rx confirmation link. Five visible steps, stepper bar, flat bundled pricing, Rx never blocks payment.
